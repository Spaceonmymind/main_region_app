# Запуск приветственного чат-бота

Проект содержит простой сценарий чат-бота MAX:

- приветствует пользователя при начале диалога;
- отвечает на команду `/start`;
- объясняет назначение мини-приложения;
- показывает кнопку «Открыть сервисы».

## Подготовка

1. Создайте бота в MAX для партнёров и дождитесь прохождения модерации.
2. Получите токен в разделе:
   `Чат-боты → Перейти → Расширенные настройки → Настроить`.
3. Скопируйте `.env.example` в `.env`.
4. Заполните `BOT_TOKEN`.
5. В настройках бота привяжите HTTPS-адрес собранного мини-приложения.
6. Заполните `MINI_APP_URL` диплинком:
   `https://max.ru/<имя_бота>?startapp`.

Этот диплинк открывает привязанное к боту мини-приложение внутри MAX.

## Запуск

```bash
yarn bot:start
```

Для локальной разработки официальная библиотека использует получение обновлений
от MAX. Для постоянной production-работы рекомендуется настроить HTTPS webhook.
